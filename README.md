## 🏛️ Project Vision
**BODHA** (Sanskrit: *बोध* - "Awakening/Perception") is a premium spiritual engineering platform. It bridges 5,000-year-old Vedic scriptures with cutting-edge Neural Networks. BODHA is designed to provide **contextual, ritualistic, and visual clarity** for modern seekers of Truth.

## 🚀 Key Features
- **Neural Shloka Decoding:** Real-time analysis of Sanskrit verses with deep-root grammatical logic.
- **Vedic Vision Engine:** Generates hyper-realistic sacred art (8K) to visualize the metaphysical essence of mantras.
- **Sutra-to-Speech (STS):** High-fidelity audio synthesis optimized for rhythmic Sanskrit chanting and phonation.
- **Mandala UI/UX:** A world-class, dark-mode "Glassmorphism" interface designed for deep focus and spiritual immersion.

## 🛠️ Technology Stack
* **Core Engine:** Sattva-V4 (Proprietary Prompt Engineering on Pollinations/Flux)
* **Frontend:** Vanilla JS / CSS3 (Atomic Architecture for ultra-fast global loading)
* **Design System:** Modern Glassmorphism + Vedic Gold Palette (#EAB308)
* **Audio:** Native Web Speech API (Optimized for Vedic cadence)
